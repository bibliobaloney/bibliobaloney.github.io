About this page

Once a week or so I run a script to collect information about newly available public documents on eCCB. I store that information in a SQLite database that you can download from Github if you want to run your own queries. Because the information is collected by a script and mostly not checked by a human, I can't guarantee that it's 100% accurate, but it should be pretty close, and more accurate than what I was capturing with the old scripts (e.g. if two reasons are found in a dismissal order, both are now collected). I've been watching CCB data since the CCB was a few months old, and I've tried to make sure the script accounts for a variety of rare occurrences and edge cases. I also have it prompt for human input when the data it grabs is hard to interpret. This page collects the results of some of the queries I thought would be most interesting. You can contact me at bibliobaloney@duck.com if you have questions about other queries.

If you're looking for the CCB info that used to be here, it's been archived and will no longer be updated.
